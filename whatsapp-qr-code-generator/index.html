How to make a WhatsApp QR code

  1. Choose WhatsApp in the generator.
  2. Select the country code and enter the phone number without local punctuation.
  3. Add an optional opening message such as “Hello, I would like to book a table.”
  4. Test the preview, then download it for print or digital use.
What it contains: the code stores a standard wa.me link. Scanning opens WhatsApp but does not send the message automatically; the visitor remains in control.

Useful business applications

Add a WhatsApp QR code to product packaging, a storefront, menus, event signs, business cards, invoices, or customer-support material. A prefilled message can help identify the source, for example “Question from the summer catalogue.”

Does the code expire?

The QR image is static and has no Honest QR Code redirect, so it does not expire. It will stop being useful only if the WhatsApp number changes, the account becomes unavailable, or WhatsApp changes its public link format.

Free and inspectable

Honest QR Code generates the image in your browser. It has no account system or QR-content API. The code is public under Apache-2.0 so developers can review, modify, and self-host the project.
