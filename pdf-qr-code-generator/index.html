How to link a QR code to a PDF

  1. Upload the PDF to your website or a file host that provides a stable public HTTPS URL.
  2. Choose PDF file link and paste that URL into the generator.
  3. Scan the preview and confirm the document opens without requiring your private login.
  4. Download PNG for everyday use or SVG for high-quality printing.
Important: the QR code stores a link, not the PDF file. The code remains valid, but the document must remain available at the same address. If the PDF URL changes, a printed static code cannot be updated.

Good uses for PDF QR codes

Link restaurant menus, instruction manuals, product specifications, event programmes, property brochures, academic papers, forms, safety sheets, or annual reports. Use descriptive link text near the code so people understand what they are opening.

Print reliably

Keep a clear quiet zone around the QR pattern, maintain strong foreground/background contrast, and avoid printing too small. Test the final artwork at its real size with more than one phone before ordering a large print run.

No subscription dependency

Honest QR Code does not replace your PDF link with a redirect it controls. The image is generated locally, has no watermark, and can be used commercially. The entire generator is available on GitHub under Apache-2.0.
