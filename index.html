Straight answers

Frequently asked questions

Will my QR code really never expire?

Yes. Honest QR Code creates static codes, so the content is encoded directly in the image. There is no Honest QR Code redirect to disable later. A destination website can still move or disappear, so use a URL you control when printing long-lived materials.

Is everything free?

Yes. Every QR type, customization option, and PNG, JPG, or SVG export is free. There is no account, watermark, scan limit, or paid tier.

Do you see the information I enter?

The generator runs locally and does not transmit the QR payload. Like most hosted websites, the web server or CDN may still record ordinary page-request information such as an IP address. You can also download the source and run it locally.

Can I use the QR codes commercially?

Yes. The images you generate can be used on menus, packaging, posters, business cards, products, and other commercial materials without attribution.

Can I copy or self-host the generator?

Yes. The source code is available on GitHub under Apache-2.0. Keep the required license and third-party notices when redistributing the software.
